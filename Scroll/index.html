Cyber Basics



Malware: Harmful software like viruses and ransomware.

Firewall: Security for controlling network traffic.

VPN (Virtual Private Network): Encrypted internet connection for privacy.

Phishing: Deceptive attempts to extract sensitive information.

2FA (Two-Factor Authentication): Dual verification for added security.

Patch/Update: Software modifications for improved security.

Encryption: Converting information for confidentiality.

Antivirus Software: Detects and removes malicious software.


Cyber Threats



Malware disrupts system.

Social engineering manipulates users.

Ransomware encrypts data for extortion.

ZeroDay exploits target unknown vulnerabilities.

Password attacks seek unauthorized access.

DDoS overwhelms systems with traffic.

Phishing pretends be a trusted actor.
